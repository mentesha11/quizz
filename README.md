# quizz - Quiz React App

Visit my live portfolio: [quizzforyou.netlify.app](https://quizzforyou.netlify.app)

## Tech Stack

- **Frontend**: React.js  
- **Hosting**: Netlify  

## Run Locally

1. Clone the repo:  
   `git clone https://github.com/mentesha11/quizz.git`
2. Install dependencies:  
   `npm install`
3. Start the server:  
   `npm start`
